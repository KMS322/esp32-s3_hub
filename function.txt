상태
STATE_HUB_INIT : 허브가 처음 켜졌을때
STATE_NVS_WIFI_EXIST : NVS에 와이파이 아이디가 존재하는 경우
STATE_NVS_WIFI_NOT_EXIST : NVS에 와이파이 아이디가 존재하지 않는 경우
STATE_BLE_WAIT_ACCOUNT : ble 연결로 부터 응답을 기다리고 있는 경우 
STATE_WIFI_CONNECT_TRY : wifi 연결을 시도하는 경우
STATE_MAC_ADDRESS_CHECK : 허브가 등록되었는지 확인하는 경우
STATE_SCAN_DEVICES : 연결가능한 디바이스를 검색하는 경우
STATE_DEVICE_DATA_RECEIVE : tailing 디바이스로 부터 데이터를 받는 경우




STATE_WEBSOCKET_INIT : 웹소켓을 하기 위해 세팅하는 경우
